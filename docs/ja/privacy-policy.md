# [API Tracker and Monitor - Debug](https://apps.apple.com/app/id6787642796)

# プライバシーポリシー

**最終更新日: 2026年7月5日**

## はじめに

API Tracker（以下「当方」）は、お客様のプライバシー保護に努めています。本プライバシーポリシーは、API Tracker macOS アプリケーション（以下「本アプリ」）のご利用時における情報の取り扱いについて説明します。

## データ収集

**API Tracker は、個人データや利用情報を外部サーバーに収集、保存、送信することは一切ありません。**

お客様のすべての API リクエスト、レスポンス、ワークスペース設定、スクリプト、設定は、お客様のローカルデバイスにのみ保存されます。当方がこれらの情報にアクセスすることはできません。

### お客様のデバイスに保存されるもの

- API リクエスト履歴と設定
- レスポンスデータと Cookie
- ワークスペースとグループの構造
- スクリプト（プリリクエストおよびポストレスポンス）
- アプリケーションの設定と環境設定
- 監視設定と結果

## サードパーティサービス

本アプリは、お客様が明示的に設定した API エンドポイントとのみ通信します。分析、クラッシュレポート、トラッキングサービスは統合されていません。

## データセキュリティ

すべてのデータはお客様のローカルデバイスに保存されるため、デバイスに組み込まれたセキュリティ機能（FileVault、システム権限など）によって保護されます。macOS を最新の状態に保つことをお勧めします。

## 子供のプライバシー

本アプリは 13 歳未満の子供を対象としておらず、意図的に子供から情報を収集することはありません。

## 本ポリシーの変更

本プライバシーポリシーは随時更新される場合があります。変更はこのページに掲載され、更新日が記載されます。
