# Política de Privacidad

**Última actualización: 5 de julio de 2026**

## Introducción

API Tracker ("nosotros") se compromete a proteger su privacidad. Esta Política de Privacidad explica cómo manejamos la información cuando utiliza la aplicación macOS API Tracker ("la Aplicación").

## Recopilación de Datos

**API Tracker no recopila, almacena ni transmite ningún dato personal o información de uso a servidores externos.**

Todas sus solicitudes API, respuestas, configuraciones de espacio de trabajo, scripts y ajustes se almacenan exclusivamente en su dispositivo local. No tenemos acceso a esta información.

### Lo que permanece en su dispositivo

- Historial y configuración de solicitudes API
- Datos de respuesta y cookies
- Estructura de espacios de trabajo y grupos
- Scripts (pre-solicitud y post-respuesta)
- Configuraciones y preferencias de la aplicación
- Configuraciones y resultados de monitoreo

## Servicios de Terceros

La Aplicación se comunica únicamente con los endpoints API que usted configura explícitamente. No se integran servicios de análisis, informes de fallos o seguimiento.

## Seguridad de Datos

Dado que todos los datos residen en su dispositivo local, están protegidos por las funciones de seguridad integradas de su dispositivo (FileVault, permisos del sistema, etc.). Recomendamos mantener su sistema macOS actualizado.

## Privacidad Infantil

La Aplicación no está dirigida a menores de 13 años y no recopilamos información de niños a sabiendas.

## Cambios en esta Política

Podemos actualizar esta Política de Privacidad periódicamente. Los cambios se publicarán en esta página con una fecha de revisión actualizada.

## Contacto

Si tiene preguntas sobre esta Política de Privacidad, abra un issue en nuestro repositorio de GitHub.
