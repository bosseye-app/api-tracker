# Politique de confidentialité

**Dernière mise à jour : 5 juillet 2026**

## Introduction

API Tracker (« nous ») s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous traitons les informations lorsque vous utilisez l'application macOS API Tracker (« l'Application »).

## Collecte de données

**API Tracker ne collecte, ne stocke ni ne transmet aucune donnée personnelle ou information d'utilisation à des serveurs externes.**

Toutes vos requêtes API, réponses, configurations d'espace de travail, scripts et paramètres sont stockés exclusivement sur votre appareil local. Nous n'avons pas accès à ces informations.

### Ce qui reste sur votre appareil

- Historique et configuration des requêtes API
- Données de réponse et cookies
- Structure des espaces de travail et des groupes
- Scripts (pré-requête et post-réponse)
- Paramètres et préférences de l'application
- Configurations et résultats de surveillance

## Services tiers

L'Application communique uniquement avec les points de terminaison API que vous configurez explicitement. Aucun service d'analyse, de rapport de crash ou de suivi n'est intégré.

## Sécurité des données

Toutes les données résidant sur votre appareil local sont protégées par les fonctionnalités de sécurité intégrées de votre appareil (FileVault, permissions système, etc.). Nous vous recommandons de maintenir votre système macOS à jour.

## Vie privée des enfants

L'Application ne s'adresse pas aux enfants de moins de 13 ans et nous ne collectons pas sciemment d'informations auprès d'enfants.

## Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une date de révision.

## Contact

Pour toute question concernant cette politique, veuillez ouvrir une issue sur notre dépôt GitHub.
