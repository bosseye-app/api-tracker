# 隱私政策

**最後更新：2026 年 7 月 5 日**

## 引言

API Tracker（"我們"）致力於保護您的隱私。本隱私政策說明了您在使用 API Tracker macOS 應用程式（"本應用"）時，我們如何處理資訊。

## 資料收集

**API Tracker 不會收集、儲存或傳輸任何個人資料或使用資訊到外部伺服器。**

您的所有 API 請求、回應、工作區設定、指令碼和設定均僅儲存於您的本機裝置。我們無法存取這些資訊。

### 保留在您裝置上的內容

- API 請求歷史記錄與設定
- 回應資料與 Cookies
- 工作區與分組結構
- 指令碼（請求前與回應後）
- 應用設定與偏好
- 監控設定與結果

## 第三方服務

本應用僅與您明確設定的 API 端點通訊。應用中未整合任何分析、崩潰報告或追蹤服務。

## 資料安全

由於所有資料均位於您的本機裝置上，它們受到您裝置內建安全功能的保護（如 FileVault、系統權限等）。我們建議您保持 macOS 系統更新。

## 兒童隱私

本應用不面向 13 歲以下兒童，我們不會有意收集兒童的資訊。

## 政策變更

我們可能會不時更新本隱私政策。變更將發布在此頁面，並標註更新日期。

## 聯絡我們

如對本隱私政策有疑問，請透過 GitHub 倉庫提交 Issue。
