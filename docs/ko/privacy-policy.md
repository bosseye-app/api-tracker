# 개인정보 처리방침

**최종 업데이트: 2026년 7월 5일**

## 소개

API Tracker("당사")는 귀하의 개인정보를 보호하기 위해 최선을 다하고 있습니다. 본 개인정보 처리방침은 API Tracker macOS 애플리케이션("본 앱") 사용 시 정보 처리 방식을 설명합니다.

## 데이터 수집

**API Tracker는 개인 데이터나 사용 정보를 외부 서버에 수집, 저장 또는 전송하지 않습니다.**

귀하의 모든 API 요청, 응답, 워크스페이스 구성, 스크립트 및 설정은 귀하의 로컬 장치에만 저장됩니다. 당사는 이 정보에 접근할 수 없습니다.

### 귀하의 장치에 저장되는 항목

- API 요청 기록 및 구성
- 응답 데이터 및 쿠키
- 워크스페이스 및 그룹 구조
- 스크립트(사전 요청 및 사후 응답)
- 애플리케이션 설정 및 기본 설정
- 모니터링 구성 및 결과

## 타사 서비스

본 앱은 귀하가 명시적으로 구성한 API 엔드포인트와만 통신합니다. 분석, 충돌 보고 또는 추적 서비스는 통합되어 있지 않습니다.

## 데이터 보안

모든 데이터가 귀하의 로컬 장치에 저장되므로, 장치에 내장된 보안 기능(FileVault, 시스템 권한 등)으로 보호됩니다. macOS 시스템을 최신 상태로 유지하시는 것을 권장합니다.

## 아동 개인정보

본 앱은 13세 미만의 아동을 대상으로 하지 않으며, 의도적으로 아동으로부터 정보를 수집하지 않습니다.

## 방침 변경

본 개인정보 처리방침은 수시로 업데이트될 수 있습니다. 변경 사항은 업데이트된 날짜와 함께 이 페이지에 게시됩니다.
