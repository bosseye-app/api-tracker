# [API Tracker and Monitor - Debug](https://apps.apple.com/app/id6787642796)

# 이용 약관

**최종 업데이트: 2026년 7월 5일**

## 제1조 약관 동의

API Tracker("본 앱")를 다운로드, 설치 또는 사용함으로써 귀하는 본 이용 약관에 구속되는 것에 동의합니다. 동의하지 않는 경우 본 앱을 사용하지 마십시오.

## 제2조 서비스 설명

API Tracker는 API 설계, 테스트 및 모니터링을 위한 macOS 애플리케이션입니다. HTTP 요청 구축, 워크스페이스 관리, 스크립트 작성 및 cURL 명령 가져오기 도구를 제공합니다.

## 제3조 라이선스

본 앱은 독점 라이선스 하에 제공됩니다. 관련 저장소에 게시된 오픈 소스 편집기 구성 요소는 MIT 라이선스 하에 제공됩니다.

## 제4조 사용자 책임

- 귀하는 본 앱을 통해 전송된 모든 API 요청에 대한 책임을 집니다.
- 귀하는 상호 작용하는 타사 API의 이용 약관을 준수해야 합니다.
- 귀하는 본 앱을 불법적인 목적이나 관련 법률을 위반하여 사용해서는 안 됩니다.
- 귀하는 장치 및 본 앱에서 사용되는 자격 증명의 보안을 유지할 책임이 있습니다.

## 제5조 인앱 구매

본 앱은 구독 플랜 및 평생 라이선스를 포함한 프리미엄 기능을 인앱 구매를 통해 제공할 수 있습니다. 모든 구매는 Apple App Store를 통해 처리되며 Apple의 약관 및 환불 정책이 적용됩니다.

## 제6조 보증 면책

본 앱은 "있는 그대로" 제공되며, 명시적이든 묵시적이든 어떠한 종류의 보증도 제공되지 않습니다.

## 제7조 책임 제한

개발자는 본 앱의 사용 또는 사용 불가로 인해 발생하는 직접적, 간접적, 부수적, 특별 또는 결과적 손해에 대해 어떠한 경우에도 책임을 지지 않습니다.

## 제8조 약관 변경

당사는 언제든지 본 약관을 수정할 권리를 보유합니다. 변경 후 본 앱을 계속 사용하면 새로운 약관에 동의한 것으로 간주됩니다.

## 제9조 준거법

본 약관은 귀하의 관할 지역에서 적용되는 법률의 적용을 받습니다.

## 제10조 문의

본 약관에 관한 질문이 있으시면 GitHub 저장소에 Issue를 생성해 주십시오.
