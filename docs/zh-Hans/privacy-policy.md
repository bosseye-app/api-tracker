# 隐私政策

**最后更新：2026 年 7 月 5 日**

## 引言

API Tracker（"我们"）致力于保护您的隐私。本隐私政策说明了您在使用 API Tracker macOS 应用程序（"本应用"）时，我们如何处理信息。

## 数据收集

**API Tracker 不会收集、存储或传输任何个人数据或使用信息到外部服务器。**

您的所有 API 请求、响应、工作区配置、脚本和设置均仅存储于您的本地设备。我们无法访问这些信息。

### 保留在您设备上的内容

- API 请求历史记录与配置
- 响应数据与 Cookies
- 工作区与分组结构
- 脚本（请求前与响应后）
- 应用设置与偏好
- 监控配置与结果

## 第三方服务

本应用仅与您明确配置的 API 端点通信。应用中未集成任何分析、崩溃报告或追踪服务。

## 数据安全

由于所有数据均位于您的本地设备上，它们受到您设备内置安全功能的保护（如 FileVault、系统权限等）。我们建议您保持 macOS 系统更新。

## 儿童隐私

本应用不面向 13 岁以下儿童，我们不会有意收集儿童的信息。

## 政策变更

我们可能会不时更新本隐私政策。变更将发布在此页面，并标注更新日期。

## 联系我们

如对本隐私政策有疑问，请通过 GitHub 仓库提交 Issue。
